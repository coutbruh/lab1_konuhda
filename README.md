Lab1: Определение типа треугольника и координат вершин
Программа принимает длины трёх сторон, определяет тип треугольника и вычисляет координаты вершин для отрисовки в поле 100x100 px.

Проверка корректности ввода (положительные вещественные числа)
Определение типа: равносторонний, равнобедренный, разносторонний, не треугольник
Расчёт координат вершин с масштабированием под 100x100
Логирование в файл и консоль
Использование
Запустите программу, введите три стороны через Enter.
